Welcome to the JoBase reference manual. Please look at the examples below before you carry on.

All colours are lists of three or four values. The values are red, green, blue and alpha. With these values, you can create any colour. The values are decimals and they range between 0 and 1. Whenever you set the colour of an object, you can pass a list, tuple or dictionary. 

                    color = [1, 0, 0] # red
                    color = (0.5, 0, 1) # purple
                    color = {"red": 1, "green": 1, "blue": 0} # yellow
                    color = (0, 1, 0, 0.5) # translucent green

All positions or sizes are made of two values - an x and a y. These values are units in pixels from the centre of the window. Just like colours, you can pass a list, tuple or dictionary when you set the position or size of an object.

class Window(caption = "JoBase", width = 640, height = 480, color = (1, 1, 1))
This class is for managing the main window. The inbuilt window variable is an instance of this class. Do not try to instance this class directly. 
                    from JoBase import *

                    window.caption = "Window Example"
                    window.color = (0, 1, 0.2)
                    window.size = (300, 300)

                    def loop():
                        window.red += 0.01

                    run()
caption: str
The title of the window.
red: float
The red color of the window.
green: float
The green color of the window.
blue: float
The blue color of the window.
alpha: float
The transparency of the window.
color: (float, float, float, float)
The color of the window.
width: float
The width of the window in pixels.
height: float
The height of the window in pixels.
size: (float, float)
The dimensions of the window in pixels.
top: float
The top of the window in pixels.
bottom: float
The bottom of the window in pixels.
left: float
The left of the window in pixels.
right: float
The right of the window in pixels.
close()
Close the window. The program finishes and the window cannot be opened again.
maximize()
Maximize the window. You can restore the maximized window with the restore method.
minimize()
Minimize the window. You can restore the minimized window with the restore method.
restore()
Restore the window from maximized or minimized. The window returns to its normal state.
focus()
Give the window input focus. It can be very disruptive to the user when the window is forced to the top - use this method sensibly.
class Camera()
This class is for managing the game camera. By moving it around, you can change your view of the objects on the screen. The inbuilt camera variable is an instance of this class. Do not try to instance this class directly. x: float
The x position of the camera.
y: float
The y position of the camera.
position: (float, float)
The position of the camera.
