First Steps

To get started with JoBase, you can use the online editor here. When you are ready, follow the steps on the homepage to install JoBase on your local computer.

At the start of every JoBase application, you need to import JoBase. Start by typing the following code. 

‡from JoBase import *‡

This imports and initialises all the tools from the JoBase library. When you run the code, it should print Welcome to JoBase in the Python Shell. Now we need to display the window for our game. 

                from JoBase import *

                ‡run()‡

The run() method displays the window and activates a game loop. When you run the code, you should have a blank, resisable window. These two lines of code are necessary for every JoBase application.

All games run inside a loop; this loop runs continuously for every frame. The code below shows how you can create this loop. 

                from JoBase import *

                ‡print("start")

                def loop():
                    print("loop")‡

                run()

You may notice that we print "start" outside the loop and we print "loop" inside the loop. When you run the code, the Python Shell will keep printing until you close the program. Now we can start drawing things on the screen. 

                from JoBase import *

                text = Text("Hello World")

                def loop():
                    text.draw()

                run()

This will render a Hello World message on the screen. Remember to always draw things in the game loop so that they are displayed every frame.

Now you understand the basics of a JoBase application, you can create your first game. If you are familiar with Python, please use the reference to get an understanding of the features of JoBase.
