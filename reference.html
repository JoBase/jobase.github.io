Whenever you come accross a size, color or position like the one below, it may seem like a tuple object. Although we have displayed it as a tuple, it behaves quite differently.

position: (float, float) The variable above is actually a JoBase Vector. This means that its properties are keys. The example below illustrates how this variable can be used. 
                ‡position.x = 2
                position.y = 3
                print(position) # (2, 3)

                position = [4, 0]
                position.y = 6
                print(position) # (4, 6)‡
JoBase Vectors also support rich comparisons and numerical operations. 
                ‡a = (5, 7)
                b = [5, 7]
                print(a == b) # True
                
                position = (5, 7)
                position += 1
                print(position) # (6, 8)

                position = (4, 1)
                position *= 2
                print(position) # (8, 2)‡
class Window(caption = "JoBase", width = 640, height = 480, color = (1, 1, 1))
This class is for managing the main window. The inbuilt window variable is an instance of this class. Do not try to instance this class directly. 
                    ‡from JoBase import *

                    window.caption = "Window Example"
                    window.color = (0, 1, 0.2)
                    window.size = (300, 300)

                    def loop():
                        window.red += 0.01

                    run()‡

The transparency of the window.

The height of the window in pixels.

The dimensions of the window in pixels.

The top of the window in pixels.

The bottom of the window in pixels.

The left of the window in pixels.

The right of the window in pixels.
close()
Close the window. The program finishes and the window cannot be opened again.
maximize()
Maximize the window. You can restore the maximized window with the restore method.
minimize()
Minimize the window. You can restore the minimized window with the restore method.
restore()
Restore the window from maximized or minimized. The window returns to its normal state.
focus()
Give the window input focus. It can be very disruptive to the user when the window is forced to the top - use this method sensibly.
class Camera()

class Key()
This class is for detecting keyboard input. The inbuilt key variable is an instance of this class. Do not try to instance this class directly. press: bool
A key is pressed down.
release: bool
A key is released.
repeat: bool
When the user holds a key down, a series of events are fired. These events do not occur every frame.
[name]: {"press": bool, "release": bool, "repeat": bool} or False
Access the state of a specific key. 
                        ‡if key.left:
                            # the left arrow key is held down

                            if key.left.release:
                                # the left arrow key is released

                        if key.right and key.right.press:
                            # the right arrow key was pressed down‡

class Shape()
This is the base class for all drawing primitives. The attributes of this class are shared with all other shapes. Do not try to instance this class directly. x, pos.x, position.x: float
The x position of the shape.
y, pos.y, position.y: float
The y position of the shape.
position: (float, float)
The position of the shape.
scale.x: float
The horizontal scale of the shape.
scale.y: float
The vertical scale of the shape.
scale: (float, float)
The scale of the shape.
anchor.x: float
The horizontal origin of the shape.
anchor.y: float
The vertical origin of the shape.
anchor: (float, float)
The rotational origin of the shape.
angle: float
The angle of the shape.
red, color.red: float
The red color of the shape.
green, color.green: float
The green color of the shape.
blue, color.blue: float
The blue color of the shape.
alpha, color.alpha: float
The transparency of the shape.
color: (float, float, float, float)
The color of the shape.
look_at(other: object)
Rotate the shape to look at another object. The object must be a Shape instance or the cursor.
move_toward(other: object, speed = 1)
Move the shape toward another object at a certain speed. The object must be a Shape instance or the cursor.

class Image(name = MAN, x = 0, y = 0, width = 0, height = 0, angle = 0, color = (0, 0, 0))
This class is for drawing images. Because it extends the Rectangle class, it contains all the same attributes. The name is a string of the filepath to the image. JoBase contains the following built in images.
MAN COIN ENEMY
The example below loads the coin image and draws it. 
                    ‡coin = Image(COIN)

                    def loop():
                        coin.draw()‡
class Text(content = "Text", x = 0, y = 0, font_size = 50, angle = 0, color = (0, 0, 0), font = DEFAULT)
This class is for drawing text messages. Because it extends the Rectangle class, it contains all the same attributes. content: str
The string content of the text.
font: str
The filepath to the font family of the text. JoBase contains the following built in fonts.
CODE DEFAULT HANDWRITING JOINED PENCIL SERIF TYPEWRITER
The example below loads the pencil font and draws the text. 
                        ‡text = Text("Hello", font = PENCIL)

                        def loop():
                            text.draw()‡
font_size: float
The size of the font.
def random(a: float, b: float) -> float
Find a random decimal between a and b.
def run()
Activate the main game loop and display the window. Always call this method at the bottom of your code.
