Squirtcopter

python -m JoBase.examples.squirtcopter
Use the arrow keys WASD or ZQSD to control the player through an infinite world. Reach the checkpoints and upgrade your helicopter.

Blue Hue

python -m JoBase.examples.blue_hue
A platform game by Joachim Ford. Use the arrow keys WASD or ZQSD to control the player around each level.
