JoBase Fast Python Game Library

Welcome to JoBase

JoBase is aimed at beginner coders who want to have fun making games with Python.

Follow steps below to prepare your computer for JoBase.

Download Python

The instructions below will show you how to set up Python on Windows.

Find the {Python Downloads} page or click the link here. On the screen you will see a yellow download button. Click on it and save the file.

When you have downloaded the file, run it. A new window should appear. BEFORE you press the {Install Now} button, tick the box that says {Add Python to PATH}. Your screen should look like this. Once you have done that, click {Install Now} and then {YES}. Python will start installing. This may take a while. Once Python has been successfully installed, you can take a break.

Install JoBase

Now it's time to install JoBase.

Open the {Command Terminal} on your computer. It's a black screen with white text. You can find it by typing {cmd} into your computer's search bar. Once you've opened it, type the following command into the terminal. 
pip install JoBase
Press {ENTER} and wait for JoBase to install. The command terminal should look similar to this. Because all operating systems are different, your command terminal may vary from the image.

If there aren't any errors (shown in red) then JoBase has been sucessfully installed.

If you have problems when installing JoBase, you may need to run the command terminal as administrator. To do this, open the command terminal by right clicking and selecting {Run as administrator} instead of just clicking on it.

If you are still struggling to set up JoBase, then ask a friend to help.

You have completed all the installation steps.
